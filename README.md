# A+ Prep Hub

CompTIA A+ study app built with React + Vite.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel deployment

This project is ready for Vercel.

Recommended settings:
- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

`vercel.json` includes a rewrite so direct navigation to app routes works correctly.
